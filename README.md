# PASSlE
My project for school

So, it use these packages:
> art
> getpass
> simple_term_menu (linux_only)

To use this programm you have to install these packages. I usually use venv not to make my computer full of different trash (don't mind, I usually do such remark)
After that you have to download access.py and main.py The latter must be choosen depending on your system because simple_term_menu library doesn't work on windows.

So, you use linux for linux and mac and windows version for (unexpected, wow) windows.
